## 2. Smoke E2E corrido (todo verde)

1. **UPDATE status='sent'** → trigger `lock_quotation_on_sent` (is_locked=true, valid_until=now+30d ✓) + `log_status_change` (audit `draft → sent` ✓) + `sync_opp` (no-op porque opp=NULL ✓).
2. **`get_public_quotation` x2** → view_count incrementó (0→1→2), viewed_at seteado ✓.
3. **`accept_public_quotation`** → status=`client_approved`, client_approved_at, audit `sent → client_approved`, in-app notif al admin con title/body/action_url ✓, WhatsApp `payment_request_v1` encolado en `notification_queue` (queda `failed` hasta que Felipe apruebe el template en Meta).
4. **Cotización final**: `client_approved` + `quotation_number=COT-2026-0008` (auto-generado por la RPC `send_quotation_to_client` cuando es la primera vez).
