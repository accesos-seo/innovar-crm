## Templates Meta pendientes — desbloqueo en una operación

Cuando Felipe apruebe + agencia (Vault) tenga los secrets Meta, el desbloqueo es 2 comandos:

```sql
-- 1) Re-encolar filas que el worker marcó failed por template_not_found
UPDATE public.notification_queue
SET status = 'pending', failed_reason = NULL, error_message = NULL, attempt_count = 0
WHERE status = 'failed'
  AND failed_reason ILIKE '%no registrado%';
```

```bash
# 2) Disparar el worker manual (o esperar al próximo cron)
curl -X POST https://xdzbjptozeqcbnaqhtye.supabase.co/functions/v1/process-whatsapp-notifications \
  -H "Authorization: Bearer $SUPABASE_SERVICE_KEY" \
  -H "Content-Type: application/json" \
  -d '{"dry_run": false, "limit": 100}'
```
